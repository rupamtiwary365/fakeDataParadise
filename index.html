About

The webpage was built during the Peace Out Hacks Hackathon hosted by MLH from 17th - 19th Sept, 2021. It generates fake csv data using faker.js javascript module. It is simple, modern looking and easy to use.

How I built it

I used HTML, CSS, Bootstrap and JavaScript for frontend. For generating data I used faker.js browser version.
